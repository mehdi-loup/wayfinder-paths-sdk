# Boros Carry Demo

Use this skill when you need to inspect or run the Boros carry demo path.

## What this skill does
- Explains the Boros carry and funding-vs-underlying backtests.
- Points to the main scripts and static applet assets.
- Helps run local checks before publishing or previewing the path.

## Steps
1. Verify the path manifest and generated applet assets.
2. Inspect the relevant backtest script for the requested workflow.
3. Run the requested command or preview the applet.
4. Summarize the output and any follow-up changes.
