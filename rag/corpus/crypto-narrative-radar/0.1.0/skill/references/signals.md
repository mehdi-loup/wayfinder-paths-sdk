# Signals

Every pipeline output must use the standard response envelope:
- `signal_snapshot`
- `selected_playbook`
- `candidate_expressions`
- `null_state`
- `risk_checks`
- `job`
- `next_invalidation`

Operational signal vocabulary:
- `armed`
- `entered`
- `exited`
- `paused`
- `null-state-selected`
- `error`
